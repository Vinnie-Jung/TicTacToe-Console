<h1 align="center">TicTacToe (Console)</h1>

<p align="center">Simple Tic-Tac-Toe console game.</p>


README: [Versão em português](https://github.com/Vinnie-Jung/TicTacToe-Console/blob/main/README.md)   
FRAMEWORK: [.NET 5.0](https://docs.microsoft.com/pt-br/dotnet/core/whats-new/dotnet-5)

<h3>About</h3>

<p>Simple tic-tac-toe project made on console using .NET for C#. The aim of the project is to practice and learn the basic/intermediate mechanisms of the language and train programming logic. The project has features such as arrays, classes and objects, methods (functions and procedures), repetition structures, conditional branch structures, etc.</p>

- Using [System](https://docs.microsoft.com/pt-br/dotnet/api/system?view=net-6.0);
- Using [System.IO](https://docs.microsoft.com/pt-br/dotnet/api/system.io?view=net-6.0);

<h3>Rules</h3>

>1) The game takes place on a 3x3 board with two players.
>
>2) One player is "X" and the other is "O". Players take turns placing their marks (X and O) on the board.
>
>3) The goal is to make a sequence of three identical symbols, either in a vertical, horizontal or diagonal line, while trying to prevent your opponent from doing the same.
>
>4) If all squares on the board are filled and none of the players managed to make a sequence of three equal symbols, the round ends in a tie.
